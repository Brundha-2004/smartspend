Authentication Endpoints
POST User Registration http://localhost:8080/auth/register
Register a new user account. Requires email, password, first name, and last name.
POST User Login http://localhost:8080/auth/login
Authenticate user and receive JWT token for subsequent requests.
GET Email Verification http://localhost:8080/auth/verify?token=VERIFICATION_TOKEN
Verify user email address using the token sent during registration.
